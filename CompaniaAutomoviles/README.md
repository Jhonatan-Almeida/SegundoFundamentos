
Sistema automatizado para el cálculo de comisiones de vendedores en una compañía automotriz. Calcula comisiones mensuales basadas en ventas de diferentes modelos de vehículos con porcentajes variables.

## Características

-  **Cálculo automáticode comisiones por modelo
-  **Múltiples porcentajes de comisión (3%, 5%, 7%)
-  **Análisis comparativo entre modelos
-  **Desglose detallado de ventas y comisiones
-  **Detección automática del modelo más rentable
-  **Validación de entradas numéricas
-  **Simulación rápida con datos de ejemplo
