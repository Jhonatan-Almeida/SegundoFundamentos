
Sistema de análisis de tiempos y velocidades para la competencia de en las Olimpiadas de Invierno. Calcula automáticamente velocidades medias y genera rankings basados en los tiempos de los atletas.

## Características

-  **Ingreso de tiempos en formato minuto:segundo:centésima
-  **Conversión automática a segundos totales
-  **Cálculo de velocidad media en m/s
-  **Generación de ranking automático
-  **Análisis comparativo entre atletas
-  **Resultados formateados
-  **Validación de datos de entrada

##  Requisitos

- **Sistema operativo:** Cualquier sistema con Python
