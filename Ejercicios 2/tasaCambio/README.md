

Una aplicación de conversión de divisas desarrollada en Python que permite cambiar Euros a Dólares aplicando tasas de cambio y comisiones de gestión.

## Características

- **Conversión Euros a Dólares
- **Cálculo automático de comisiones (10% por defecto)
- **Desglose detallado de la operación
- **Historial de operaciones
- **Interfaz intuitiva con menús interactivos
- **Validación de entradas de usuario

## Requisitos

- **Sistema operativo:** Windows, Linux o macOS
