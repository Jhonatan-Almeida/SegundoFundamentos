

Script en Python que duplica cada carácter de un string de entrada. Ideal para ejercicios de manipulación de strings y práctica de programación funcional.
 Características

    Duplicación precisa de cada carácter

    Validación estricta de longitud de entrada

    Múltiples métodos de implementación

    Optimizado para Python 3.13 con características modernas

    Código limpio y bien documentado

Ejemplos

Entrada: 'sbc56'
Salida:  'ssbbcc5566'
