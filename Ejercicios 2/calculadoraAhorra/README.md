# Calculadora de Ahorros Anuales

Una aplicación en Python para calcular y analizar los ahorros anuales basándose en los ingresos y gastos del usuario.

## Descripción

Este programa permite:
- Calcular ganancias anuales basadas en el salario por hora y horas trabajadas
- Determinar gastos anuales a partir de los gastos semanales
- Calcular los ahorros anuales netos
- Analizar escenarios alternativos (como trabajo a tiempo parcial)
- Evaluar la viabilidad financiera de diferentes situaciones

##  Características

### Funcionalidades principales:
1. **Cálculo de ingresos**: Salario semanal y anual
2. **Cálculo de gastos**: Gastos semanales y anuales  
3. **Cálculo de ahorros**: Diferencia entre ingresos y gastos
4. **Análisis de escenarios**: Comparación entre situación actual y tiempo parcial
5. **Validación de datos**: Entrada segura y manejo de errores

### Escenario especial incluido:
- Trabajo a tiempo parcial (25 horas semanales)
- Reducción de gastos al 75%
- Análisis de viabilidad financiera


##  Requisitos

- **Python 3.6** o superior
- No se requieren librerías externas
