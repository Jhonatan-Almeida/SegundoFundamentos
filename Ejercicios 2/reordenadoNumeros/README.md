
Programa en Python para practicar manipulación de números mediante descomposición en dígitos y reversión de números. Incluye dos ejercicios fundamentales para el manejo de strings y operaciones numéricas.

Características

    **Descomposición numérica: Separa números en sus dígitos individuales

     **Reversión de números: Invierte el orden de los dígitos

    **Validación robusta: Verificación de entradas numéricas

    **Múltiples métodos: Diferentes enfoques para cada problema

    **Análisis detallado: Información adicional sobre los números procesados

    **Ejemplos interactivos: Modo demostración incluido
