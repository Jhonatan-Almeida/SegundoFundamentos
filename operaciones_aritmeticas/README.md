# Operaciones Aritméticas

Este proyecto contiene scripts en Python para realizar diversas operaciones aritméticas básicas.

## Archivos incluidos

- `operaciones_aritmeticas.py` - Script principal con interfaz de usuario

## Operaciones implementadas

### a. Operación aritmética fija
Calcula el resultado de:
\[
\left( \frac{3 + 2}{2 \times 5} \right)^2
\]

### b. Suma de los primeros n números
Para un entero positivo `n` introducido por el usuario, calcula:
\[
\frac{n(n+1)}{2}
\]

### c. División con cociente y resto
Para dos números enteros introducidos por el usuario, muestra:
- Los números de entrada
- El cociente de la división
- El resto de la división


## Ejecución

### Script principal:
```bash
python operaciones_aritmeticas.py
