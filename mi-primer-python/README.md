# Ejercicio Python  de Bienvenidad 

Este es mi primer proyecto en Python donde practico la sintaxis básica, manipulación de strings.

## Objetivo
Crear un script que dé la bienvenida, formateando correctamente mi nombre, sin importar cómo lo introduzco.


##  Funcionalidades

- Almacenar y mostrar strings en variables
- Solicitar el nombre mediante input()
- Convertir texto a mayúsculas y minúsculas
- Formatear nombres (primera letra mayúscula, resto minúsculas)
- Eliminar puntos accidentalmente introducidos
- Mensaje final con formato
